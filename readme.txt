To begin 
-> In this directory create 2 folders named:
	1) marked
	2) to_mark
-> Copy task folders into "to_mark" folder
-> When prompted enter status and comment
-> When complete a "marks" csv will appear in the "marked" folder
-> The program moves marked pdfs into "marked" folder
-> Each new entry is saved in "marked.csv" in the "marked" folder
-> After an entry has been made the relvant row is removed from "to_mark.csv"
